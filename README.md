# multi_thread_queue_cpp 

multi_thread_queue_cpp is the resolution of an exercise where two threads access an Queue class

## Installation

Use CMake to install the required libraries and necessary files

```bash
mkdir build
cd build
cmake ..
```

## Usage

```bash
make 
make test # optional
./multi_thread_queue_cpp
```
